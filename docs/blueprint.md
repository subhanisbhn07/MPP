# **App Name**: Compare Phones

## Core Features:

- Mobile Search: Users can search for phones by name, brand, or specifications.
- Phone Comparison: Users can compare up to four phones side-by-side.
- Homepage Listing: Browse and list popular, latest or discounted mobile phones on the homepage based on some parameters.
- AI Spec Generation Tool: AI tool to programmatically generate specification details, that admin users can manually add or edit.
- Wishlist: Users can add phones to their wishlist for later viewing.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke trust and reliability, suitable for a tech-focused comparison site.
- Background color: Light gray (#F0F2F5) to provide a clean and neutral backdrop.
- Accent color: A vivid purple (#9C27B0) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque sans-serif font for a modern, objective feel.
- Simple and modern line icons to represent phone features and specifications.
- Clean and responsive grid layout optimized for mobile and desktop devices.
- Subtle transition animations to improve user experience when navigating between phone comparisons and search results.
